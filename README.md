# Iniciar en modo desarrollo

1. Clonar el repositorio
2. Instalar Nest-CLI
```
npm i -g @nestjs/cli
```
3. Ejecutar el comando
```
yarn install
```
4. Levantar la base de datos MongoDB
```
docker-compose up -d
```

## Stack usado
* MongoDB
* Nest
